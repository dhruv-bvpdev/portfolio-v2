2. Works Page
3. Work Detail Page

### Usage

#### Install Dependencies

- yarn install

#### Install Dependencies (sanity)

- cd sanity
- yarn install

#### Run (localhost:3000)

- yarn dev

#### Run Sanity (localhost:3333)

- cd sanity
- sanity start

#### Sanity Deployment (localhost:3333)

- cd sanity
- sanity deploy
